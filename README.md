## 🔬 Test Principles

### PT (Prothrombin Time)
- **Pathway:** Extrinsic (Tissue Factor → Factor VII → X → II → Fibrin)
- **Application:** Warfarin monitoring (INR 2.0-3.0)
- **Detection:** Optical transmission method

### APPT (Activated Partial Thromboplastin Time)
- **Pathway:** Intrinsic (Contact Activator → XII → XI → IX+VIII → X → II → Fibrin)
- **Application:** Heparin monitoring, Lupus anticoagulant detection
- **Detection:** Kinetic analysis with curve response

### Fibrinogen (Clauss Method)
- **Principle:** Direct thrombin addition → clot formation time
- **Application:** Fibrinogen deficiency assessment, DIC monitoring
- **Normal Range:** 2.0-4.0 g/L

### D-Dimer & FDP
- **Detection:** Immunoturbidimetry
- **Application:** Thrombosis/DIC screening, Fibrinolysis monitoring
- **D-Dimer:** <0.5 mg/L (normal)

### AT-III
- **Detection:** Chromogenic substrate method
- **Application:** Heparin resistance assessment, Thrombosis risk evaluation
- **Normal Range:** 80-120%

## 💡 Clinical Applications

**Coagulation Assessment**
- Pre-operative bleeding risk evaluation
- Anticoagulation therapy monitoring (Warfarin, Heparin)
- Factor deficiency diagnosis (Hemophilia A, B, etc.)

**Anticoagulant Monitoring**
- INR monitoring for Warfarin
- aPTT monitoring for Heparin
- Direct thrombin inhibitor monitoring

**Special Testing**
- Lupus anticoagulant detection
- Inhibitor screening (Factor VIII, etc.)
- DIC diagnosis and monitoring

**Fibrinolysis Assessment**
- D-Dimer: Thromboembolism screening (high NPV)
- FDP: DIC and fibrinolysis monitoring
- Thrombus stability evaluation
